# 점수 집계 기능 백로그

> 향후 추가할 점수 요소들과 구현 계획

---

## 1. 목요일 라이브 개근 (팀별)

### 개요
- **목적**: 라이브 세션 참여 독려
- **단위**: 팀별
- **난이도**: ⭐ 쉬움

### 데이터 입력 방식
**Option A**: Google Form 출석 체크
- 라이브 중 QR/링크로 출석 폼 제출
- 자동으로 시트에 기록

**Option B**: 수동 입력
- 운영진이 직접 출석 인원 입력

### 스프레드시트 구조
```
새 시트: "라이브 출석"

| 날짜       | A팀 출석 | B팀 출석 | C팀 출석 | D팀 출석 | E팀 출석 | F팀 출석 |
|-----------|---------|---------|---------|---------|---------|---------|
| 2026-01-09 | 10      | 12      | 8       | 14      | 9       | 11      |
| 2026-01-16 | 12      | 14      | 10      | 12      | 10      | 10      |
| ...        |         |         |         |         |         |         |
```

### 점수 계산 로직
```
점수 집계 시트에 추가할 열:
- "라이브 출석": 해당 팀의 총 출석 횟수 합계
- "라이브 보너스": 전원 출석 주차 × 100점

수식 예시 (N열 - 라이브 보너스):
=SUMPRODUCT(('라이브 출석'!B2:B10 >= 소그룹 팀원수) * 1) * 100
```

### 대시보드 반영
- 점수 집계 CSV에 자동 포함
- 상세 순위표에 "라이브" 열 추가

## 2. 톡방 프로그램 참여 (팀별)

### 개요
- **목적**: 톡방 내 미션/이벤트 참여 독려
- **단위**: 팀별
- **난이도**: ⭐ 쉬움

### 데이터 입력 방식
- 운영진이 프로그램별 참여 여부 체크

### 스프레드시트 구조
```
새 시트: "프로그램 참여"

| 프로그램명      | 배점 | A팀 | B팀 | C팀 | D팀 | E팀 | F팀 |
|---------------|-----|-----|-----|-----|-----|-----|-----|
| 자기소개 미션   | 50  | ✓   | ✓   | ✓   |     | ✓   | ✓   |
| 인증샷 챌린지   | 30  | ✓   |     | ✓   | ✓   |     | ✓   |
| 피드백 릴레이   | 40  |     | ✓   | ✓   | ✓   | ✓   |     |
```

### 점수 계산 로직
```
점수 집계 시트에 추가할 열:
- "프로그램 점수": 참여한 프로그램 배점 합계

수식 예시 (O열 - A팀 프로그램 점수):
=SUMPRODUCT(('프로그램 참여'!C2:C10=TRUE) * '프로그램 참여'!B2:B10)

또는 체크박스 대신 1/0 사용:
=SUMPRODUCT('프로그램 참여'!C2:C10 * '프로그램 참여'!B2:B10)
```

### 대시보드 반영
- 상세 순위표에 "프로그램" 열 추가
- 또는 보너스에 합산

## 3. 톡방 내 최고 조회수 가산점

### 개요
- **목적**: 고성과 콘텐츠 제작 독려
- **단위**: 팀별 (팀 내 최고 조회수 기준)
- **난이도**: ⭐⭐ 중간

### 데이터 입력 방식
**Option A**: 과제 폼에 조회수 필드 추가
- 제출 시점 조회수 입력
- 단점: 제출 시점마다 다름

**Option B**: 별도 조회수 기록 시트
- 특정 시점(예: 주차 마감일)에 일괄 기록
- 장점: 공정한 비교 가능

### 스프레드시트 구조
```
새 시트: "조회수 기록"

| 이름   | 팀 | 1주차 조회수 | 2주차 조회수 | 3주차 조회수 |
|-------|-----|------------|------------|------------|
| 홍길동 | A   | 1,200      | 2,500      | 800        |
| 김철수 | A   | 500        | 1,800      | 3,200      |
| ...   |     |            |            |            |
```

### 점수 계산 로직
```
점수 집계 시트에 추가할 열:
- "최고 조회수": 팀 내 최고 조회수
- "조회수 점수": 구간별 점수 또는 비례 점수

수식 예시 (팀별 최고 조회수):
=MAX(FILTER('조회수 기록'!C2:C100, '조회수 기록'!B2:B100=A2))

구간별 점수 예시:
=IFS(
  최고조회수 >= 10000, 100,
  최고조회수 >= 5000, 70,
  최고조회수 >= 1000, 40,
  최고조회수 >= 500, 20,
  TRUE, 0
)
```

### 대시보드 반영
- 상세 순위표에 "조회수" 열 추가
- 최고 조회수 달성자 하이라이트 (선택)

## 4. 팔로워 증가 가산점

### 개요
- **목적**: 계정 성장 독려
- **단위**: 팀별 (팀 평균 또는 합계)
- **난이도**: ⭐⭐ 중간

### 데이터 입력 방식
- 챌린지 시작 시 팔로워 수 기록
- 종료 시 (또는 주기적으로) 현재 팔로워 수 기록

### 스프레드시트 구조
```
새 시트: "팔로워 변화"

| 이름   | 팀 | 시작 팔로워 | 현재 팔로워 | 증가량 | 증가율 |
|-------|-----|-----------|-----------|-------|-------|
| 홍길동 | A   | 150       | 320       | 170   | 113%  |
| 김철수 | A   | 1,200     | 1,450     | 250   | 21%   |
| ...   |     |           |           |       |       |

* 증가량 = 현재 - 시작
* 증가율 = (현재 - 시작) / 시작 * 100
```

### 점수 계산 로직
```
점수 집계 시트에 추가할 열:
- "팔로워 증가": 팀 평균 증가량 또는 합계
- "성장 점수": 가중치 적용 점수

수식 예시 (팀 평균 증가량):
=AVERAGEIF('팔로워 변화'!B2:B100, A2, '팔로워 변화'!E2:E100)

점수 변환 (증가량 10당 1점):
=ROUND(팀평균증가량 / 10, 0)

또는 증가율 기반:
=AVERAGEIF('팔로워 변화'!B2:B100, A2, '팔로워 변화'!F2:F100) * 가중치
```

### 고려사항
- 시작 팔로워가 적은 사람이 유리할 수 있음 (증가율 기준 시)
- 시작 팔로워가 많은 사람이 유리할 수 있음 (증가량 기준 시)
- **제안**: 증가량 + 증가율 복합 점수 또는 구간별 점수

### 대시보드 반영
- 상세 순위표에 "성장" 열 추가
- 팀별 평균 증가량 표시

| 1 | 라이브 개근 | 기존 보너스 구조와 유사, 바로 적용 가능 |
